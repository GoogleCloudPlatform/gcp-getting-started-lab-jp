# VPN接続方法・トラブルシューティングガイド

## 1. 接続できない場合の確認事項
- インターネットに接続されていますか？
- GlobalProtectアプリは最新ですか？
- パスワードの有効期限は切れていませんか？

## 2. エラーコード別対処法
- **Error 1001**: 認証サーバーに接続できません。社内ネットワーク担当に連絡してください。
- **Error 2020**: 証明書の期限切れです。ITポータルから新しい証明書をダウンロードしてください。

## 3. 初期設定手順
1. ポータルサイト (vpn.example.com) にアクセス
2. 社員IDとパスワードでログイン
3. クライアントアプリをダウンロードしてインストール
4. サーバーアドレスに `vpn.example.com` を入力
