# Migration to GCP

This extension provides an AI-powered migration assistant for moving workloads from AWS to Google Cloud Platform.

## Skills

- **aws-to-gcp**: Full migration workflow — discover AWS infrastructure, clarify requirements, design GCP architecture, estimate costs, and generate Terraform + migration artifacts.

## MCP Servers

- **gcloud**: Official Google MCP server (`@google-cloud/gcloud-mcp`) — runs gcloud CLI commands for GCP resource management and billing queries.
- **google-developer-knowledge**: Official Google Developer Knowledge MCP server (`https://developerknowledge.googleapis.com/mcp`) — searches official Google developer documentation for architecture validation, product lifecycle checks, and current Google Cloud docs.

## Usage

Say "migrate from AWS to GCP" or "AWS to Google Cloud migration" to activate the migration skill.
